# Origin OS

Origin OS is a community-driven custom ROM project built on top of LineageOS.

The project aims to provide a clean, reliable, and long-term software experience for Nothing devices, especially after official software support ends.

## Vision

Modern smartphones often remain capable long after official updates stop.

Origin OS exists to extend the life of supported devices by delivering:

* Continued Android updates
* Security patches
* Long-term maintenance
* A refined user experience
* Community-driven development

## Current Focus

### Nothing Phone (1)

Initial development is focused on the Nothing Phone (1) (spacewar).

Goals:

* Android 16 base
* LineageOS 23 foundation
* Stability-first approach
* Nothing-inspired user experience
* Long-term support

## Roadmap

### Phase 1

* Integrate Nothing applications

* Community testing

### Phase 2

* Origin OS Alpha
* User interface improvements

### Phase 3

* OTA update infrastructure

### Future Support

As official support for Nothing devices reaches end-of-life, Origin OS aims to provide a path forward for the community.

Potential future targets include:

* Nothing Phone (2)
* Nothing Phone (3a)
* Nothing Phone (3)
* Additional supported devices

Device support decisions will depend on:

* Community interest
* Available maintainers
* Hardware compatibility

## Philosophy

Origin OS is not intended to replace LineageOS.

Instead, it builds upon the excellent foundation provided by the LineageOS project while delivering an experience tailored to supported devices.

Core principles:

* Stability before features
* Long-term maintainability
* Open development
* Community collaboration

Origin OS is currently in the planning and foundation stage.
